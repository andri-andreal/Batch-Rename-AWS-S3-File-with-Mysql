# Batch Renamer AWS S3 file with Mysql (PYTHON 3)


## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install requirement plugin. (This case use python3)

```bash
pip install -r requirement.txt
```

## Usage

```bash
python3 main.py
```
